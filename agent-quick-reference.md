## üõ†Ô∏è Troubleshooting

### When Agent Doesn't Understand

**Instead of:** "Handle my stuff"
**Try:** "Show me today's tasks and urgent emails"

**Instead of:** "Email thing"
**Try:** "Search emails from Sarah about the proposal"

**Instead of:** "Fix it"
**Try:** "Reschedule overdue tasks to next week"

### Getting Better Results

‚úÖ **Be specific:** Include names, dates, topics
‚úÖ **Use natural language:** "tomorrow", "next Monday"
‚úÖ **Confirm important actions:** Review before sending emails
‚úÖ **Provide context:** Reference previous conversations

## üö® Important Notes

**Privacy:**
- Agent never shares API keys
- Confirms before sending emails
- Asks permission for bulk operations (>10 items)

**Limitations:**
- Can't access emails older than Gmail API allows
- Todoist API rate limits may apply
- Works best with specific, clear requests

**Best Practices:**
- Review generated tasks before creating
- Confirm email drafts before sending
- Provide feedback to improve accuracy
- Start simple, add complexity gradually
